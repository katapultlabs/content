# Katapult Context

## What We Are
Software development agency doing long-term team augmentation for complex product engineering. Development, design, product ownership, management, and fractional C-level work when appropriate.

## Language Rules
- **Partners, not clients.** Everyone Katapult works with is a partner. We work in partnerships to build great things together. Never refer to "clients" in any Katapult content.
- This applies to all written output: articles, case studies, proposals, decks, emails, social posts.

## Voice
All content uses Voice DNA plugin. Key rules: no em dashes, no fatal pattern (negation-assertion), short paragraphs, physical verbs, contractions always. See Voice DNA skill for full rules.
